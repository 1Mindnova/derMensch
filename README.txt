Projekt: fiin-a-uman – Surah Al-Insan
Dieses Projekt synchronisiert Verse des Koran-Kapitels 'Der Mensch (Al-Insan)' mit einer Audioaufnahme.

Inhalt:
- index.html: Hauptseite
- style.css: Design und Schrift
- script.js: Steuerung der Anzeige synchron zur Audio
- verses.json: Zeitlich getaktete Versdaten
- audio.mp3: Audio (bitte sicherstellen, dass die Datei vorhanden ist)

Anleitung:
1. Alle Dateien in dein GitHub-Repository (z. B. 1Mindnova/derMensch) hochladen.
2. Stelle sicher, dass 'audio.mp3' im selben Ordner liegt.
3. Aktiviere GitHub Pages (Settings → Pages → Deploy from branch → main/(root)).
4. Öffne die Projektseite und klicke auf ▶ Play.
